# Links


## 1. Soft Links

- soft links are like shortcuts in windows, when created they have a diferent inode from the original file.

- create soft link :
  ```
  ln -s targetFile sourceFile
  ```


## 2. Hard Links
- Hard links share the same inode of the original file.

- create hard link :
  ```
  ln -s targetFile sourceFile
  ```
